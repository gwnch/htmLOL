The Basics



At its most simplistic, propositions are statements that can either be true or false. An example of a propositional statement in English is: If the sun shines, then the birds start singing.


This is likely true, unless you have some weird birds in your town. As well as expressing propositions in plain text, they can be translated into mathematical representation.

To mathematically represent the proposition “If the sun shines, then the birds start singing” we first must translate the parts of the proposition into logical variables. Let’s use s and b, for sun and birds respectively. Now we will introduce the symbols that will help us complete the mathematical proposition.


Name Symbol Meaning
Negation ′ or ~ not
Conjunction ∧ and
Disjunction ∨ (inclusive) or
XOR ⊕ (exclusive) either/or
Conditional ⇒ if/then, implies, only if, is sufficient for, is necessary for
Biconditional ⇔ if and only if, is necessary and sufficient for, is logically equivalent to


Our proposition is a conditional if/then statement, meaning that b is only true if s is true, if the sun shines. So we can represent the proposition mathematically as: s⇒b.

Tables

Truth tables allow us to explore the relationship between the different parts of a proposition. A truth table for our proposition would be:

Parse Trees

The proposition s⇒b is pretty simple, but expressions can become much more complicated, including more than two variables and many operations. Our textbook provides a complex propositional expression: [(p∨q)∧((p′)∨r)]⇒[(p⇔q)∨(p∨r)]. If a propositional truth table contains all truths down the final column of the table, it is considered a tautology. If there are all falses, it is a contradiction.

Application

It is easier to place importance on things learned in school if there is some direct real world application in addition to the valuation of knowledge for knowledge’s sake. Propositional logic is vital to computer hardware, since computer circuits are built using logic gates. The very operations we discussed above—not, and, or—are the ones used in logic gates. Pictured below are representations of these logic gates.

The T and F of the truth tables and parse trees are converted into 1s and 0s, the language of computers. These gates can be combined to perform all sorts of operations. Information that is inputted yields information at an output which gives the computer a command, operating similar to a switch.
